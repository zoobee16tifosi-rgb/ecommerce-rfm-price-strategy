## 1. 프로젝트 개요

**목적**: 이커머스 유통사 판매 데이터를 분석하여 고객 세그먼트와 가격 전략을 통합한 데이터 기반 매출 증대 방안 도출

**핵심 문제**:
* 획일적 마케팅: 시즌·성별·연령별 수요 차이를 반영하지 못한 일괄 마케팅
* 카테고리별 가격 불균형: 시장 대비 극단적으로 분산된 가격 포지셔닝
* 낮은 재구매율: 전체 재구매율 1.90%로 매우 낮은 수준

**핵심 가설**:
* 가설①: 시즌별 트렌드와 고객 세그먼트를 결합하면 전환율과 재구매율이 상승한다
* 가설②: 가격대를 3단계(프리미엄/중가/저가)로 다층화하면 매출과 객단가가 함께 증가한다

**팀 구성**:

| 역할 | 담당자 | 담당 업무 |

| 팀장 | 박주비 | 세그먼트 분석 / PPT&발표 / 총괄 |
| 팀원 | 익명 | 세그먼트 분류&분석 / 보고서 / Tableau |

## 3. 가설① 세그먼트 전략

### 세그먼트 설계

기존 마케팅 수신 동의 여부 기반 분류의 한계를 확인 (t-test p-value = 0.9588)  
→ 실제 구매 행동 데이터 기반 **Trend × Value 2차원 매트릭스**로 재설계

**1차 분류: RFM 기반 고객 가치 (4단계)**

| 세그먼트 | 기준 (R/M) | 비율 |
| --- | --- | --- |
| High Value | R≥4 AND M≥4 | 16.55% |
| Mid-High Value | R≥3 AND M≥3 | 20.89% |
| Mid-Low Value | 나머지 | 47.26% |
| Low Value | R≤2 AND M≤2 | 15.30% |

**2차 분류: Trend Sensitivity 트렌드 민감도 (3단계)**  
네이버 쇼핑 인사이트 API로 수집한 월별 검색량 Z-score 활용

| 세그먼트 | 기준 (z-score) | 비율 |
| --- | --- | --- |
| Trend Hunter | z ≥ 1 | 12.95% |
| Core | 0 ≤ z < 1 또는 NaN | 28.74% |
| Value Seeker | z < 0 | 58.31% |

**최종 세그먼트**: 4단계 × 3단계 = **12개 세그먼트**

## 4. 가설② 가격대 확장 전략

### 현황 및 문제

* 카테고리별 가격이 시장 대비 0.3~3.4배로 극단 분산
* **프리미엄 라인 부재**: 30~50대 고지출층 니즈 미충족
* **저가 라인 부재**: 10~20대 가격 민감층 유입 한계

### 프리미엄 라인 분석 (상향 전략)

평가 공식: `기대매출(R) = 가격 상승폭(P) × [선호도(Q_pref) + 필요성(Q_need)]`

| 순위 | 소카테고리 | 최종 점수 | 가격 상승 여력 |
| --- | --- | --- | --- |
| 1 | 냉장고 | 55.33 | +732천원 |
| 2 | 음향가전 | 54.49 | - |
| 3 | 세탁기/건조기 | 54.26 | +666천원 |
| 4 | 청소기 | 52.22 | - |
| 5 | 캠핑 | 51.55 | - |

### 저가 라인 분석 (하향 전략)

평가 공식: `고객 유입 효과(Impact) = 타겟 규모(Volume) × 방문 빈도(Frequency)`

| 순위 | 소카테고리 | 최종 점수 | 10-20대 비중 |
| --- | --- | --- | --- |
| 1 | 메이크업 | 90.48 | 72% |
| 2 | 스킨케어 | 61.89 | - |
| 3 | 마스크/팩 | 42.05 | 46% |
| 4 | 헤어케어 | 40.27 | - |
| 5 | 선케어 | 36.86 | - |

→ 저가 라인은 플랫폼 이미지 보호를 위해 **서브 브랜드관("Young Line") 형태로 분리 운영** 제안

## 7. 핵심 교훈

* 마케팅 수신 동의 여부는 실제 구매 행동에 유의미한 영향 없음 (p-value = 0.9588) → 행동 기반 세그먼테이션이 효과적
